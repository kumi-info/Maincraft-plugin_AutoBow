# Maincraft-plus_AutoBow
【Maincraft】タワーディフェンス-自動連射

Paper 1.21 プラグイン — 指定プレイヤーが弓を持って右クリックすると自動で弓を引いて発射を繰り返す。

## インストール

`AutoBow-1.0.0.jar` を `plugins/` に配置してサーバーを再起動。

## コマンド

| コマンド | 説明 |
|---------|------|
| `/autobow hold <speed>` | 長押しモード＋速度設定 |
| `/autobow toggle <speed>` | トグルモード＋速度設定 |
| `/autobow on <player>` | 指定プレイヤーの自動弓を有効化（永続） |
| `/autobow off <player>` | 指定プレイヤーの自動弓を無効化 |
| `/autobow status` | 現在の設定状況を表示 |
| `/autobow list` | 有効中のプレイヤー一覧 |

エイリアス: `/ab`

## モード

| モード | 動作 |
|--------|------|
| hold | 右クリック長押しで連射、離すと停止 |
| toggle | 1クリックで連射開始、もう1クリックで停止 |

## 速度（speed）

数値が小さいほど速い（単位: tick）。小数点OK。最小値: 1

| speed | 間隔 | 備考 |
|-------|------|------|
| 1 | 0.05秒 | 最速 |
| 5 | 0.25秒 | 高速 |
| 10 | 0.5秒 | やや速い |
| 20 | 1秒 | デフォルト |

## 動作仕様

- 弓を引く → 発射 → リリース → 引き直す のサイクルを繰り返す
- 引き直しの間は速度に比例した自然なタイミング
- 以下の条件で自動停止:
  - holdモード: 右クリックを離す
  - toggleモード: もう一度右クリック
  - 弓/クロスボウを持ち替える
  - 矢がなくなる（クリエイティブは無限）
  - プレイヤーがログアウト

## データ永続化

- `/autobow on <player>` で有効化した設定はサーバー再起動後も保持される
- モードと速度の設定も保存される
- 保存先: `plugins/AutoBow/config.yml`

## ビルド

```
mvn clean package
```

出力: `target/AutoBow-1.0.0.jar`

要件: JDK 21, Maven
